program_03_06：

一个简单的例子，用于找出`N`以内的素数。

跟上一版本的区别：

| 3.5版本(上一版)                        | 3.6版本(这一版)                                      |
| -------------------------------------- | ---------------------------------------------------- |
| `N`在程序中写死为10000                 | `N`从参数中传来                                      |
| 数组在栈中分配内存                     | 数组在堆中分配内存，因此能计算更大的`N`              |
| 数组是通过标准数组的定义来实现的：a[N] | 数组通过针对来实现：int *a = malloc(N * sizeof(int)) |
