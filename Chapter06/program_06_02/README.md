程序6.2：选择排序

[选择排序可视化过程](https://algorithm-visualizer.org/brute-force/selection-sort)

#### 算法步骤

- 首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置
- 再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。
- 重复第二步，直到所有元素均排序完毕。
